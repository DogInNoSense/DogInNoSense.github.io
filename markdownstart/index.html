MarkDown 一些基本样式学习

发表于2022-11-04|更新于2022-11-06|markdown
|阅读量:

标题

在MarkDown的编写中，#标记为标题，写法为#号+空格+标题名。一级标题为#，二级标题为##，三级标题为### ，以此类推，最多为6级标题

字体

  1. ** +文字+** 为文字加粗，注意**与文字之间不要有空格 hello markdown
  2. *+文字+*为斜体 hello markdown
  3. *** + 文字 + *** 为斜体加粗 hello markdown
  4. ~~ 文字 ~~ 表示删除，无空格 hello markdown

引用

  • “>” 为引用，示例如下

    鲁迅说过…

分割线

  • 分割线有两种方式可产生，“***”或者连续三个“-”

图片

  • 图片的编写是![图片名](图片地址),图片地址可以是网址，亦可以是能访问系统文件的路径。

列表

有序列表“1.”+“空格 ”，无序列表“-”+“空格”

有序列表

  1. 鲁迅说过…
  2. 周树人…

无序列表

  • 这是无序列表1
  • 这是无序列表2
