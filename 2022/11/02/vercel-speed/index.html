vercel 教程

  1. 访问Vercel官网，点击右上角的sign up进行注册并登陆
  2. 注册完成后选择新建一个项目


  3. 填入你的github上部署网页的地址
    例如我的是: https://github.com/DogInNoSense/DogInNoSense.github.io

  4. 导入静态页面仓库之前，需要为你的Github安装Vercel，此处建议选择All repositories，意为为所有仓库安装，当然，你也可以选择只为当前仓库安装，也就是Only select repositorie
  5. PROJECT NAME可以任意写
  6. 可以使用Vercel提供的默认域名来访问页面了

更新 hexo 博客

修改 _config.yml文件

将地址更改为新的仓库地址即可
不过我发现访问量从头开始计了,以前的访问量被清空了
vercel加速的好处之一就是仓库可以设为private了,能够保护自己的写的代码

绑定自定义域名

前提你要有一个域名

  1. 找到域名添加页
  2. 在域名管理后台添加两条解析记录(CNAME记录和A记录)


    注意他给的记录填上去不能用,我搜到以下解决方法:

2021年5月初，Vercel因为一些原因被GFW屏蔽，导致无法访问
A记录就从76.76.21.21 改成 52.76.85.65或者76.223.126.88，CNAME记录则是从cname-vercel-dns.com.改成 cname-china.vercel-dns.com；关于A记录，可用的IP值也可以从这个里面找：https://gist.github.com/ChenYFan/fc2bd4ec1795766f2613b52ba123c0f8
添加完这两条记录后,稍等片刻即可通过自定义域名访问!
